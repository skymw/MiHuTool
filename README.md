# 项目介绍
    本项目针对以往工作中所有使用到的工具类进行汇总
    包括,时间操作工具,文件操作工具,JDBC数据访问SDK,sftp访问工具,neo4j操作工具,经纬度计算转换工具等，，
    

# 环境依赖
    jdk 1.8

     ├── ReadMe.md           // 帮助文档
     
     |──cn.skymw.muhu  //程序根目录 
     
       ├── character.CharacterUtil    // 中文转换拼音工具类
       
       ├── common             // 公共辅助类
       
       ├── files.FileUtil     // 文件(IO)流操作类
       
       │── jdbc.spring    //JDBC数据访问SDK
       
       │── msutils     //sftp 操作工具
       
       │── neo4j       //neo4j 操作工具
       
       │── postions   //坐标转换工具
       
       │── times     //时间操作
